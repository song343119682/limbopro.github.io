ClashA 配置教程

1.确认你所用的机场是否支持Clash订阅；

via n3ro

2.复制Clash 订阅链接；

3.如上图，打开ClashA，点击Profiles，点击+/ New Profile - > URL - 给该订阅命名（Name），一般为设置为机场名字，在URL处粘贴Clash订阅链接；

4.点击右上角确认保存，返回到上一个页面（如上图位置），选中任意订阅名称；
5.返回主界面；
6.开启开关；

分应用代理设置

1.通过以上的设置，其实基本上可以很好的在工作生产中使用ClashA了；
2.如需要给某些App走代理，可通过分应用代理进行设置；

设置步骤
