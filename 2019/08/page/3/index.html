主理人序

最近一段时间，自学了DDoS的一些相关知识，更确切的说应该是CC（ChallengeCollapsar)。
上 Cloudflare CDN 隐藏源站IP，引用 Nginx 模块
--with-http_realip_module 以获取 CDN 背后用户的真实 IP，使用 Ubuntu 系统下 cron服务每分钟自动运行 CC封锁 脚本，设定一定阈值，将异常请求IP拉黑名单；
