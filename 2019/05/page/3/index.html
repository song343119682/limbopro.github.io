抑郁的解释

抑郁是情绪低落、厌恶活动的一种状态，对人的思想、行为、感觉和身体健康都有一定影响。 抑郁的人可能会感到沮丧、焦虑、空虚、绝望、罪恶、烦躁、不安等情绪。 他们可能会对平时的爱好不再感兴趣，丧失食欲或暴饮暴食，变得不善交际，出现记忆力下降、难以集中注意力、容易犹豫不决等问题。
