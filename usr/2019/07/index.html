使用浏览器开发者工具纯手工下载

0.安装 uBlock Origin - 免费且卓越的广告拦截程序！

1.进入任意 PB 视频播放页面

2.谷歌浏览器右上角 打开 开发者工具，或使用快捷键 ：Mac 系统 谷歌浏览器快捷键为 alt + command +i，Windows 系统 谷歌浏览器快捷键 F12 或 Ctrl + Shift + I。

3.找到 Elements，点击它，选中Elements 下方任意内容；

4.使用快捷键寻找关键字 mp4；接上一步，按住 command + f （Mac系统）唤出 Elements 内 搜索框；或 Ctrl+f 唤出 Elements 内 搜索框；输入 关键字 mp4 进行搜索；正常情况下，有6个搜索结果，第四个结果为最高清（显著 标准 720P 480P字样；）；

5.选中刚刚搜索后的结果，鼠标右键选中 Edit as HTML
6.选中视频链接部分（如上图），右键，转到链接；
7.下载开始；
